# Регистрация пользователя
![Рисунок 2](./Безиaffcafcwмени.png)

Пример запроса:

`POST/sudent
```json
{
  "User name": "Ivan",
  "email": "ivan@gmail.com",
  "password": "53132f6ew",
}
```
Пример ответа:

```json
{
  "id_student": "1",
  "User name": "Ivan",
  "email": "ivan@gmail.com",
  "password": "53132f6ew",
  "createdAt": "2022-06-22T11:40:00Z",
  "updatedAt": "2022-06-22T11:40:00Z"
}
```
